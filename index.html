PinFetcher Bot

Finalidade: Este bot foi criado para integrar o Discord com a API oficial do Pinterest.

Ao utilizar comandos simples, usuários podem buscar imagens por palavras-chave diretamente em canais do Discord.

Funcionalidade:

  • Busca imagens públicas no Pinterest
  • Exibe as imagens automaticamente no canal de texto
  • Não armazena ou coleta nenhum dado pessoal
